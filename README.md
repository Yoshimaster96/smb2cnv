# SMB2CNV, a Super Monkey Ball 2 level converter
By Yoshimaster96

## About
This neat utility converts OBJ and MTL data, and exports a GMA, TPL and LZ model file.

## Usage
`smb2cnv [obj file] [config file]`
### Required libraries:

* None (just C standard library)

### Known bugs:

* None at the moment.
